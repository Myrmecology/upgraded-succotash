================================================================================
                        MINI PROJECTS COLLECTION
================================================================================

Welcome to the Sub Projects folder! This directory contains 19 interactive
Python mini programs, all accessible through a central launcher.

================================================================================
                             QUICK START
================================================================================

1. Make sure you have Python 3.6 or higher installed

2. Open a terminal/command prompt in this folder

3. Run the main launcher:
   
   python main.py

4. Select a program by entering its number (1-19)

5. Enjoy! Each program will return you to the main menu when you're done

================================================================================
                            FEATURES
================================================================================

* No external dependencies - uses Python standard library only
* All programs are self-contained and independent
* Easy-to-use numbered menu system
* Programs loop internally so you can play multiple times
* Automatic return to main menu when finished
* Works on Windows, Mac, and Linux

================================================================================
                         RUNNING PROGRAMS
================================================================================

METHOD 1: Use the Main Launcher (Recommended)
   python main.py

METHOD 2: Run Individual Programs
   python coin_flip.py
   python tictactoe.py
   (or any other program file)

================================================================================
                         TROUBLESHOOTING
================================================================================

Q: Program won't start
A: Make sure you're running Python 3.6 or higher. Check with: python --version

Q: Colors don't display correctly
A: Some terminals don't support ANSI colors. Try a modern terminal like 
   Windows Terminal, iTerm2, or update your terminal.

Q: Import errors when running main.py
A: Make sure all .py files are in the same folder and you're running main.py
   from within the Sub Projects directory.
